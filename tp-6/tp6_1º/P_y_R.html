Preguntas y respuestas segundo html

1-¿Qué propiedad CSS se utiliza para establecer un contenedor como contenedor flex?

Se utiliza display

2-¿Qué valor se utiliza en la propiedad `flex-wrap` para permitir que los elementos se envuelvan en líneas adicionales si no hay suficiente espacio?

El valor utilizado en la propiedad flex-wrap permite que los elementos se envuelvan en líneas adicionales si no hay suficiente espacio se escribe flex-wrap: wrap. Esto permite que los elementos se ajusten en líneas adicionales si no caben todos en una sola línea.

3-¿Cuál es la propiedad CSS que se utiliza para centrar los elementos a lo largo del eje principal horizontal en un contenedor flex?

La propiedad CSS utilizada para centrar los elementos a lo largo del eje principal horizontal en un contenedor flex es justify-content

4-¿Cuál es la propiedad CSS que se utiliza para centrar los elementos a lo largo del eje secundario vertical en un contenedor flex?

La propiedad CSS utilizada para centrar los elementos a lo largo del eje secundario vertical en un contenedor flex es align-items

5-¿Cuál es el propósito del valor `justify-content: space-between` en la propiedad `justify-content` del encabezado?

Su propósito es distribuir los elementos del encabezado de manera uniforme a lo largo del eje principal horizontal, es decir el espacio se distribuirá de manera equitativa entre los elementos, empujando el primero hacia un extremo y el último hacia el otro extremo del contenedor

6-¿Cuál es el propósito del valor `align-items: baseline` en la propiedad `align-items` del encabezado?

Su propósito es alinear los elementos del encabezado a lo largo de la línea de base, garantizando que los elementos del encabezado tengan la misma línea base

7-¿Qué propiedad CSS se utiliza para establecer el ancho del contenedor del título del header. Clase container-title ?

La propiedad CSS para establecer el ancho del contenedor del título en el header con la clase container-title es width. Se establece como width: 100%, lo que significa que el contenedor del título ocupa todo el ancho del contenedor padre

8-¿Qué propiedad CSS se utiliza para alinear el texto del título en el centro?

La propiedad CSS utilizada para alinear el texto del título en el centro es text-align. En este caso, se establece como text-align: center para la clase .container-title

9-¿Cuál es la propiedad CSS que se utiliza para establecer un margen a la derecha y a la izquierda del contenedor?

La propiedad CSS utilizada es margin. Se establece como margin: 0 50px, lo que significa que hay un margen de 0 en la parte superior e inferior y un margen de 50px en ambos lados

10-¿Qué propiedad CSS se utiliza para establecer el color de fondo del contenedor principal?

La propiedad CSS utilizada para el color de fondo es background-color. Se escribe como background-color: #ffcad4, lo que establece el color de fondo del contenedor principal en un rosado claro
